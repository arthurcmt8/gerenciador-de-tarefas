# 📋 Task Manager

A simple command-line task manager built with Python.

## Features

- Add tasks
- List all tasks
- Mark tasks as completed
- Remove tasks
- Save tasks in a JSON file

## Technologies

- Python 3
- JSON

## How to Run

1. Clone the repository.
2. Open the project folder.
3. Run:

```bash
python main.py
```
